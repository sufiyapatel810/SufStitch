# 🧶SufStitch
SufStitch is a simple e-commerce website for selling handmade crochet products like bags, accessories, and decor.

✨ Features:
1. Browse products
2. Add to cart
3. Basic checkout
4. Responsive design

🛠️ Tech Stack:
- HTML
- CSS
- JavaScript

🚀 How to Run:
1. Clone the repo
```bash
git clone https://github.com/sufiyapatel810/SufStitch.git
```
2. Open index.html in your browser
